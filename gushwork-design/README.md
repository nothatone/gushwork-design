# Gushwork Web Development Assignment

This project is created using only:

- HTML
- CSS
- JavaScript

No frameworks or libraries are used.

## Features

- Responsive design
- Maximum content width of 1240px
- Sticky header on scroll
- Image carousel
- Image zoom on hover
- FAQ accordion
- Horizontal application slider
- Contact form section
- Beginner-friendly line-by-line code

## Files

- index.html
- styles.css
- script.js
- assets/images

## How to Run

Open `index.html` in your browser.
